# 📊 Employee Attrition Analysis Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard that analyzes employee attrition using HR data. The report helps identify employee turnover patterns based on education, city, gender, payment tier, and experience in the current domain. It also compares overall attrition counts with attrition rates to provide deeper business insights.

---

## 🎯 Project Objectives

- Analyze employee attrition using HR data.
- Identify employee groups with higher attrition.
- Build an interactive dashboard for HR decision-making.
- Compare attrition count and attrition rate for better analysis.

## 💡 Key Findings

### Executive Dashboard (Count-Based)

- Total Employees: **4,653**
- Employees Left: **1,600**
- Attrition Rate: **34.39%**
- Highest Attrition Count observed in **Bachelor's Degree** employees.
- Highest Attrition Count observed in **Pune**.
- Highest Attrition Count observed in **Payment Tier 3**.

### Advanced Analysis (Rate-Based)

- Highest Attrition Rate observed among **Master's Degree** employees.
- **New Delhi** records the highest attrition rate.
- **Payment Tier 2** has the highest attrition rate.
- **Female** employees show a higher attrition rate.
- Employees with the highest attrition rate can be identified using the experience analysis chart.
